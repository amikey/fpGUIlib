Here demo of fpGUI library used by Java.

There is the fpGUI wrapper class: => fpg.java
And the demo main class: => fpg_demo.java

Add Java library libfpgui.so or fpgui.dll into /fpGUIlib/demos/demo_Java

-In terminal do:

-> cd /demo_Java   (go in java demo path)

-> java -Djava.library.path=. fpg_demo

This all. :-)
